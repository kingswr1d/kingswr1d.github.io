Detection Engineering Lab — Sigma + MITRE ATT&CK
github.com/kingswr1d/soc-lab-termux/detection
Active

A detection engineering pipeline built on Sigma rules, validated empirically against Atomic Red Team test cases and mapped to MITRE ATT&CK. Rules compile to Splunk SPL, Elastic EQL, Microsoft Sentinel (KQL), and Google Chronicle via sigmac — making them production-ready across major SIEM platforms.

rules/*.yml
Sigma rules covering Execution, Credential Access, Persistence & Defense Evasion (T1059, T1003, T1053)
test_rules.py
Runs each Sigma rule against relevant Atomic Red Team techniques and outputs match counts to test_results.json
coverage_map.py
Generates a MITRE Navigator layer JSON from current rule coverage — drag & drop into ATT&CK Navigator
mitre_coverage_layer.json
Visual ATT&CK coverage map showing detected vs gap techniques — used in interviews and reports
